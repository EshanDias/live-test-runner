# v2.0.0 Implementation Progress

**Version decision:** v2.0.0 — replacing native VS Code Testing API with fully custom webview UI.  
**Plan source:** Plans/v1.1.0.md  
**Started:** 2026-04-03

---

## Why v2.0.0?
The plan replaces the entire `vscode.TestController` / `TestRun` API with custom `WebviewViewProvider` panels. This is a breaking architectural change — the native Test Results panel is gone, replaced by our own sidebar + panel views.

---

Each feature area gets its own branch, merged to `develop`, then `main` on release.
